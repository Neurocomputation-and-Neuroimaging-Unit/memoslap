# MeMoSLAP
Repository for code specific to the [MeMoSLAP](https://www.memoslap.de/en/home/) project

### Neuronavigation
   - everything related to neuronavigation with the Nexstim system (eXimia software, Polaris sensors)
   - transformations between coordinates systems
   - current simulations:
        - target placement optimization for neuronavigation
        - (access to the memoslap_utils from a git repository in Greifwald required)

### Data upload to XNAT
   - importing data from the CCNB Dicom server to XNAT server of MeMoSLAP

### Data analysis on High Performance Cluster
   - fMRI analyses scripts for memoslap project based on the NNU batch scripts
   - for batching it on HPC

### Automatic Reminder Emails for Appointments 
   - script for checking calendar for upcoming appointments and sending participants reminder emails
